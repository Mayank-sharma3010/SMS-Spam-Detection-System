# 📱 Advanced SMS Spam Detection System

An advanced Machine Learning based SMS Spam Detection System developed using **Java 17**, **Spring Boot**, **Weka**, **MySQL**, **Spring Security (JWT)**, and **Thymeleaf**.

The application classifies SMS messages as **Spam** or **Ham (Not Spam)** using Natural Language Processing (NLP) and Machine Learning techniques.

# ▶️ Run the Project

1. Clone the repository.
2. Open the project in VS Code or IntelliJ IDEA.
3. Configure MySQL in `application.properties`.
4. Update Maven dependencies.
5. Run `SpamDetectorApplication.java`.
6. Open your browser and access the application.
